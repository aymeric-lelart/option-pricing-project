# Option Pricing Project

This project implements a European option pricer using the Black-Scholes model, including full Greeks computation and visualisation.


## Features

- European Call & Put pricing (Black-Scholes closed-form)
- Payoff vs market price visualisation
- Full Greeks dashboard:
  - **Delta** — sensitivity of option price to underlying price moves
  - **Gamma** — rate of change of Delta (convexity)
  - **Vega** — sensitivity to volatility (per 1% move)
  - **Theta** — time decay (per calendar day)
  - **Rho** — sensitivity to interest rate (per 1% move)
- At-the-money Greeks summary table
